# Mario 專案座標與場景設定紀錄（2x版本）

## 1. 座標系統
- 原點：左上角 `(0,0)`
- `x` 向右增加
- `y` 向下增加

---

## 2. 原始背景資訊
- 原始背景尺寸：`3376 x 480`

---

## 3. 背景縮放設定
- 背景縮放倍率：`2x`
- 縮放後背景尺寸：
  - Width: `3376 × 2 = 6752`
  - Height: `480 × 2 = 960`

---

## 4. 畫面顯示範圍（地面區域）

原本只顯示上半部：
- `y = 0 ~ 240`

放大 `2x` 後：
- `y = 0 ~ 480`

說明：
- `240 × 2 = 480`
- 對應原本「地面全景」

---

## 5. 地板線（Ground）

原始：
- `groundY = 208.96`

放大 2 倍：
- `groundY = 208.96 × 2 = 417.92`

整數建議：
- `groundY = 418`

---

## 6. Mario 角色尺寸（不縮放）

- Width: `32`
- Height: `64`

說明：
- 背景放大
- Mario 不縮放 → 相對變小

---

## 7. Mario 起始位置

### 原始
- `spawnX = 120`
- `groundY = 208.96`

---

### 放大 2 倍後

- `spawnX = 120 × 2 = 240`
- `groundY = 417.92`

---

### 左上角座標（關鍵）

```txt
playerX = 240
playerY = groundY - 64
        = 417.92 - 64
        = 353.92
